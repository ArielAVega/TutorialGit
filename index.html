Bienvenidos al Tutorial de Git

Para iniciar, debemos empezar a seguir este proyecto, por lo que debemos ejecutar el comando git init

Esto creará la carpeta .git

El proyecto es nuestra área de trabajo, mientras que la carpeta .git mantiene los snapshots o instántaneas que serían como fotos del proyecto. Cada una de estas fotos constituye una versión del proyecto
